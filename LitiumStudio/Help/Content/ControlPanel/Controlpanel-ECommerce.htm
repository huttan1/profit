Litium Studio E-Commerce

General settings

General settings allows customizing display columns available in the order list and delivery list. By default few columns will be selected to be displayed in these lists but you can select the display fields according to your requirement.

 

Select “Control Panel” in the Litium Studio menu found in the top left corner. Then select “E-Commerce” in the left menu to display configuration options for Litium Studio E-Commerce.

Customize order list columns

In order to add new display columns.

  1. Select column(s) from the “Order list - Available fields” panel. In order to select multiple fields click on the required fields while pressing on CTRL key.
  2. Click Add. The selected fields will be moved to the “Order list - Visible fields” panel.

 

Display columns will be given in the order in which it is placed in “Order list - Visible fields” in the List. Use the arrows to rearrange the fields.

 

Click “Save” to update the changes.

Customize delivery list columns

In order to add new display columns.

  1. Select column(s) from the “Delivery list - Available fields” panel. In order to select multiple fields click on the required fields while pressing on CTRL key.
  2. Click Add. The selected fields will be moved to the “Delivery list - Visible fields” panel.

 

Display columns will be given in the order in which it is placed in “Delivery list - Visible fields” in the List. Use the arrows to rearrange the fields.

 

Click “Save” to update the changes.

Orders

Settings for orders allows you to configure order management related settings.

Editing orders manually
  1. Select “Enable editing orders manually” checkbox.
  2. Click Save.

 

Once enabled any signed in user, with permissions, will be able to edit/modify existing orders.

Creating orders manually
  1. Select Enable creating orders manually checkbox.
  2. Click “Save”.

Once enabled any signed in user, with permissions, will be able to manually create new orders.

Shopping chart

This section allows defining settings related to shopping chart functionality available in your public site.

 

Specify saving option. Available options are

  • Do not save: If you select this option shopping cart details will not be saved for any type of user (anonymous user/Registered user). When the browser are closed, all details will be gone.
  • Save for anonymous user: This option will save shopping chart details for all users.
  • Save for logged in user: This option will only save shopping chart details of signed in users (registered users).

 

Click “Save” to update modifications.

Translations

Translations enables you to provide language translations for available order/payment/delivery statuses and payment/delivery methods. All available order status will be listed in the “Order statuses” panel and the languages you have enabled in the application will be displayed in form of columns.

 

Click “Save” to save the modifications.

Delivery methods

This section allows you to create, edit and delete delivery methods used in order management. A delivery method have a description and VAT connected. The delivery price will then be calculated in “Delivery plug-in”, so modifications needs to be done there as well.

Add delivery methods

By default pre defined delivery methods will be created during installation. But if required you can also create new delivery methods in the following manner

 

  1. Click “New delivery method”.
  2. Enter following details.
    Description - Delivery description.
    Delivery VAT% - VAT % charged for the delivery method.
    Delivery provider - If this method should be connected to a specific delivery provide, please choose from the drop down list. All available providers will be shown.

 

Click “Save” to save details.

Click “Cancel” to abort changes made.

Edit delivery method
  1. Select the delivery method which you want to edit.
  2. Click “Edit”.
  3. Enter following details.
    Description - Delivery description.
    Delivery VAT% - VAT % charged for the delivery method.
    Delivery provider - If this method should be connected to a specific delivery provide, please choose from the drop down list. All available providers will be shown

 

Click “Save” to save.

Click “Cancel” to abort changes made.

Delete delivery method
  1. Select the delivery method which you want to remove.
  2. Click “Delete”.
  3. Click “Ok” to the confirmation message that follows. The selected delivery method will be removed permanently.

Delivery providers

The usage of a delivery provider is a way to communicate with external companies, such as DHL, Schenker etc.

Create a delivery provider
  1. In order to add a delivery provider, click “New delivery provider” found in the toolbar.
  2. Enter following details.
    Delivery provider name - The display name to use.
    Delivery provider setting path - Add the path to the user control (ascx-file) that will be shown when clicking “Send to delivery provider”, found when editing deliveries.

 

Click “Save” to save details.

Edit delivery provider
  1. Select the delivery provider which you want to edit and click “Edit”.
  2. Enter following details.
    Delivery provider name - The display name to use.
    Delivery provider setting path - Add the path to the user control (ascx-file) that will be shown when clicking “Send to delivery provider”, found when editing deliveries.

 

Click “Save” to save modifications.

Delete delivery provider
  1. Select the delivery provider which you want to remove and click “Delete”.
  2. Click “Ok” to the confirmation message that follows. The selected delivery provider will be removed permanently.

Campaign types

Settings for campaign types enables you to give translations for the campaign types available in the Litium Studio E-Commerce section.

 

All campaign types will be listed here and the languages enabled in Litium Studio will be given as separate columns.

Provide translations for the available campaign types accordingly.

 

Click “Save” to save modifications.

Reports

This section is dedicated to the management of reports.

A report consists of a part that needs to be programmed. That task needs to be done before a report can be added here.

Add report
  1. Click “New report”.
  2. Enter following details
    Active - Specify if the report is active by selecting this checkbox. Only active reports will be displayed in report section.
    Name - Name of the report.
    Translations - All languages enabled in the application will be given along with a text field. Provide translations for report name here.
    Path to user defined reports - Path to the user control (ascx-file) that will be displayed.

 

Click “Save” to save details.

Edit report
  1. Select the report to be modified and click “Edit”.
  2. Enter following details
    Active - Specify if the report is active by selecting this checkbox. Only active reports will be displayed in report section.
    Name - Name of the report.
    Translations - All languages enabled in the application will be given along with a text field. Provide translations for report name here.
    Path to user defined reports - Path to the user control (ascx-file) that will be displayed.

 

Click “Save” to update modifications.

Delete reports
  1. Select the report which you want to delete and click “Delete”.
  2. Click “Ok” to the confirmation message that follows. The selected report will be deleted permanently.
