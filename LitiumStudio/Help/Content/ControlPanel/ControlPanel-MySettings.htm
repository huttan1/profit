My settings

Select “Control Panel” in the Litium Studio menu found in the top left corner. Then select “My settings” in the left menu to display settings for the current user.

 

 

General

It exists two possible configurations:

  • Skin - Select which skin (Theme) that should be inside inside Litium Studio administration.
  • Language - The current user language are selected from the drop down list.

Password

In this section it is possible to change the password of current user.

 

This section is not visible if you are signed in as “System” user.
