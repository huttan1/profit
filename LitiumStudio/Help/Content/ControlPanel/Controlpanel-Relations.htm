Litium Studio Relations

It exists several configuration possibilities for Litium Studio Relations. Some of them are needed to be filled in before using this application.

Select “Control Panel” in the Litium Studio menu found in the top left corner. Then select “Relations” in the left menu to display configuration options for Litium Studio Relations.

General settings

Litium Studio allows you to choose which fields should be visible in the person list view.

All available text fields will be listed in the “Available fields” panel.

 

In order to add fields to the list, select the required field from the list and click add.

The added fields will be displayed in the “Visible fields” in the list panel. Ensure that you add the field in which order you need them to be displayed. Fields will be displayed in that order.

 

Click “Save” to save modifications.

Field templates

Field templates defines what kind of information that can be stored by an object that uses that templates.

Create a field template

Select “New field template” found in the toolbar.

 

Enter following details

  • Name - Specify the name of the template here.
  • Template type - Select the template type. E.g. person, group or organization. As soon as the template field is selected it gets disabled. Hence if you require changing the template type you have to cancel creation and start from the beginning.

Once the template is selected following system defined fields will be displayed to you.

Fields displayed here will depend on the selected template type. Hence only fields relevant to the template will be displayed here.

 

  • Description - Description of the template.
  • Settings for fields: All system specified fields will be listed here. Specify which of these has to displayed, hidden and mandatory by selecting the options given in the drop down.
    Select “Visible and required” for fields which are important and must be specified during the creation of a person, group, etc.
    Select “Visible” for fields which should be visible but are not considered mandatory.
    Select “Hidden” to hide the fields which are mpt needed to be displayed in the interface.
  • Add definition field - You can attach the required user defined fields to the template from here. All user defined fields created in will be given in the “Add definition field” drop down. The user defined fields are created under the page “Fields” found in settings for Litium Studio Relations.
  • Setting for custom templates - Provides you the possibility to customize your own views when editing, viewing or adding a person, group or organization that uses this field template.

 

Click “Save” create the template.

Click “Cancel” to abort changes made.

View template

Select the template you wish to view and click “View”.

 

Click “Back” to return to field template list.

Click “Edit” to modify the selected template.

Click “Delete” to delete the selected template.

Edit template

Select the template you wish to edit and click “Edit”.

 

You will not be able to edit the “template type”. Thus if you require changing the template type you have to delete the template.

 

Enter following details

  • Name - Specify the name of the template here.
  • Description - Description of the template.
  • Settings for fields: All system specified fields will be listed here. Specify which of these has to displayed, hidden and mandatory by selecting the options given in the drop down.
    Select “Visible and required” for fields which are important and must be specified during the creation of a person, group, etc.
    Select “Visible” for fields which should be visible but are not considered mandatory.
    Select “Hidden” to hide the fields which are mpt needed to be displayed in the interface.
  • Add definition field - You can attach the required user defined fields to the template from here. All user defined fields created in will be given in the “Add definition field” drop down. The user defined fields are created under the page “Fields” found in settings for Litium Studio Relations.
  • Setting for custom templates - Provides you the possibility to customize your own views when editing, viewing or adding a person, group or organization that uses this field template.

 

Click “Save” to save changes.

Click “Cancel” to abort changes made.

 

Delete template

Select the template you wish to remove and click “Delete”.

 

Click Ok to the confirmation message that follows. Selected template will be removed permanently.It is not possible to remove a template which is been used in creating existing persons/groups/organizations.

 

Fields

Except for the predefined fields that are specific for each template type, it is possible to define custom fields. When a custom field are created, it is then possible to attach it to several field templates.

Create field

Select “New Field” in the toolbar.

 

Enter following details

  • Name - Name of the field.
  • Type - Type of the field. Select a suitable field type from the drop down. E.g. date, file, image.

 

Once the type is defined several settings will be visible.

  • Administration - Specify how the particular field should be displayed in the administration by selecting any of the options given in the drop down.
    “Visible and editable” - This option will display the field and will also allow modifications.
    “Hidden” - This option will hide the field in Litium Studio. A good solution if the field are used in an integration or if the value are used inside code.
    “Visible and uneditable” - The field will be visible but will not allow modifications. It will be readonly.
  • My Page - Specify how the particular field should be displayed on the web, on “My Page”, by selecting any of the options given in the drop down.
    “Visible and editable” - This option will display the field and will also allow modifications.
    “Hidden” - This option will hide the field in Litium Studio. A good solution if the field are used in an integration or if the value are used inside code.
    “Visible and uneditable” - The field will be visible but will not allow modifications. It will be readonly.
  • Mandatory - Select this checkbox to make the field mandatory so that a value must be entered in this field.
  • Alternative: Select this checkbox to specify alternative values for the field. Enter alternative values in the text box given.

 

Click “Save” to save the field.

Click “Cancel” to abort changes made.

Edit field

Select a field from the list and click “Edit”

 

Enter following details

  • Name - Name of the field.
  • Administration - Specify how the particular field should be displayed in the administration by selecting any of the options given in the drop down.
    “Visible and editable” - This option will display the field and will also allow modifications.
    “Hidden” - This option will hide the field in Litium Studio. A good solution if the field are used in an integration or if the value are used inside code.
    “Visible and uneditable” - The field will be visible but will not allow modifications. It will be readonly.
  • My Page - Specify how the particular field should be displayed on the web, on “My Page”, by selecting any of the options given in the drop down.
    “Visible and editable” - This option will display the field and will also allow modifications.
    “Hidden” - This option will hide the field in Litium Studio. A good solution if the field are used in an integration or if the value are used inside code.
    “Visible and uneditable” - The field will be visible but will not allow modifications. It will be readonly.
  • Mandatory - Select this checkbox to make the field mandatory so that a value must be entered in this field.
  • Alternative: Select this checkbox to specify alternative values for the field. Enter alternative values in the text box given

 

Click “Save” to save the field.

Click “Cancel” to abort changes made.

Delete field

Select the field which you wish to delete and click “Delete” Click “OK” to the confirmation message that follows. The selected template field will be deleted permanently.

Roles

This tab allows you to create application user roles and also see how many users that are assigned as that role.

Create a role

Click “New role” in the toolbar.

 

Enter following details.

  • Role name - Give the role name here.
  • Description - Description of the template.

 

Click “Save” to save the role.

Click “Cancel” to abort changes made.

View role

Select a role from the list and click “View”.

Details about the role will be displayed. All persons that are assigned to that role will appear in a list.

 

Click “Back” to return to the page listing the roles.

Click “Edit” to edit the selected role details.

Click “Delete” to delete the selected role.

Edit role

Select the role to be modified and click “Edit”.

 

Enter following details.

  • Role name - Give the role name here.
  • Description - Description of the template.

 

Click “Save” to save the role.

Click “Cancel” to abort changes made.

Delete role

Select the user role to be deleted.

Click Delete.

Click Ok to the confirmation message that follows. The selected role will be deleted permanently.
