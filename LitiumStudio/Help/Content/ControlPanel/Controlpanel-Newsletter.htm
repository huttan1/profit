Litium Studio Newsletter

It exists several configuration possibilities for Litium Studio Newsletter. Some of them are needed to be filled in before using this application.

Select “Control Panel” in the Litium Studio menu found in the top left corner. Then select “Newsletter” in the left menu to display configuration options for Litium Studio Newsletter.

General settings

This page contains all settings needed to setup Litium Studio Newsletter.

Sender name

This name is used as default sender name when creating circulars but can be changed when new circulars is created.

Sender address

This e-mail address is used as default sender address but can be changed when new circulars are created.

Path to pickup directory

This path is used by the server to pick up circulars and send them. Default is "c:\inetpub\mailroot\pickup"

Path to server

This path is used to show an address to the circulars online-version. It needs to be exactly as the domain that is registred. Include "http://", example "http://www.litium.se"

Character encoding

E-mail messages are always encoded with a character set. The most recent standard is UTF-8, but some older e-mail programs may have difficulties to present these messages properly. For messages built with western languages ISO-8859-1 will be compatible with most number of clients.

Text version

This text is used as default text for recipients that are not able to read HTML-mail. To include the address to the online version of the circular use ##OnlineVersionAddress##, and this will be replaced when sending.

Circular templates

Litium Studio provides you with the possiblity to use specific designed circular templates.

Select “Templates” found in the left menu under “Circular” node.

Create a new template

Select “New template” in the toolbar. In the next page it will be possible to add information about the new guide.

Enter following details:

  • Name - Name of the template.
  • Description - Brief description of the template.
  • Template File - Path to the location where the template file is saved.
    E.g. Site\Newsletter\Templates\Base2.aspx
  • Thumbnail Image - Path to the location where the template image is saved
    E.g. Site\Newsletter\Templates\Images\Base2.gif

 

Click “Save” to save the template.

Click “Cancel” to abort changes made.

Edit circular template

Select a template from the list and click “Edit”, found in the toolbar.

Edit following details:

  • Name - Name of the template.
  • Description - Brief description of the template.
  • Template File - Path to the location where the template file is saved.
    E.g. Site\Newsletter\Templates\Base2.aspx
  • Thumbnail Image - Path to the location where the template image is saved
    E.g. Site\Newsletter\Templates\Images\Base2.gif

 

Click “Save” to save the template.

Click “Cancel” to abort changes made.

Delete circular template

Select the template to be removed and click “Delete” in the toolbar.
