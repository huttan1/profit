Litium Studio Media Archive

It exists two configuration possibilities in Litium Studio Media Archive.

  • One possibility is metadata. Metadata is description about data. They not only provide details but also tag media files with information so that they can be searched easily. Metadata created in settings will be available for all files uploaded. Hence it is advisable to only create common metadata fields here. Metadata fields can also be created when uploading the file to media archive.
  • The other configuration possibility is predefined versions for your images. That will allow you to setup common format for versions, such as thumbnail version(72 x 72 px).

 

Select “Control Panel” in the Litium Studio menu found in the top left corner. Then select “Media Archive” in the left menu to display configuration options for Litium Studio Media Archive.

Metadata

Select “Metadata” found in the left menu under “Media Archive” node.

Add metadata

Click “Add metadata” found in the toolbar.

Add metadata - Step 1

Enter details:

  • Metadata - Name of the metadata.

 

Click “Save” to save the new metadata,

Click “Cancel” to abort changes made.

Edit metadata

Select a metadata field from the list and click “Edit” in the toolbar.

 

Edit details:

  • Metadata - Name of the metadata.

 

Click “Save” to update modifications.

Click “Cancel” to abort changes made.

Delete metadata

Select a metadata field from the list and click “Delete” in the toolbar.

Confirm that you actually want to remove the metadata field.

Versions

Select “Versions” found in the left menu under “Media Archive” node.

Create a new version

Click “New version” found in the toolbar.

Create a new version - Step 1

Enter details:

  • Name: Name of the predefined version.
  • Two scale options exists:

    Max width/Max height
    Max width - Defined max width of the predefined version. If “Set only max width” are checked then the height of the new version can be at any size.
    Max height - Defined max width of the predefined version. If “Set only max width” are checked then the height of the new version can be at any size.
    Keep aspect ratios of the file - If this option are checked then the proportions will be kept.

    Percentage
    Define how large the new version will be in percentage, based on the old image.

Click “Save” to save the predefined version.

Click “Cancel” to abort changes made.

View predefined version

Select a version from the list and click “View” in the toolbar.

Details of the selected version will be displayed.

Click “Edit” to edit the version.

Click “Delete” to delete the version.

Click “Back to previous step” to return to edit guide.

Edit predefined version

Select a version from the list and click “Edit” in the toolbar.

Edit details:

  • Name: Name of the predefined version.
  • Two scale options exists:

    Max width/Max height
    Max width - Defined max width of the predefined version. If “Set only max width” are checked then the height of the new version can be at any size.
    Max height - Defined max width of the predefined version. If “Set only max width” are checked then the height of the new version can be at any size.
    Keep aspect ratios of the file - If this option are checked then the proportions will be kept.

    Percentage
    Define how large the new version will be in percentage, based on the old image.

 

Click “Save” to update modifications.

Click “Cancel” to return to Media Archive settings page.

Delete prefined version

Select a version from the list and click “Toolbar” in the toolbar.

Confirm that you actually want to remove the metadata field.
