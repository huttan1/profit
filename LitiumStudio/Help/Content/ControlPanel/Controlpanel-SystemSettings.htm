System settings

Select “Control Panel” in the Litium Studio menu found in the top left corner. Then select “System settings” in the left menu to display general configuration options for Litium Studio. This settings will only be described briefly and are for system administators only. More information about this settings can be found at Litium Studio Knowledge Center.

 

Groups

This area contains all user groups that are used inside Litium Studio. Each user group contains permissions for different actions. Please note that “Besökare” are the one that hold permissions for all not signed in users, visitors on the web site.

Users

All users that should be able to work inside Litium Studio should be administrated here. Please note that all not signed-in users will be identified as “Anonymous users”

Languages

All valid languages are administrated here. Every language that occurs in this section can be used as a back office language (If the texts are translated) and it can be used as languages when working inside Litium Studio Product Catalog.

Spelling dictionaries

Administrate spelling dictionaries that are used in all WYSISYG-editors inside Litium Studio. Free spelling dictionaries can be found at Open Office (www.openoffice.org).

Currencies

All currencies are managed in this section. Every currency has a currency code, a name and a rate.

Modules

This area contains settings for all modules. It also contains settings for module panels. In general this specific is for system administrators only. See Litium Studio Knowledge Center for more information.

Messages

“Messages” are an area where all translations are made for Litium Studio Back Office. This modifications are done by an administrator.

Event log

The event log contains all logged information inside Litium Studio. It can be errors or information messages. A log entry are graded from 0 - 100 where 100 are the most critical.

License

This area contains information about the current license. It consists of several parameters that are retrieved from the license file. See Litium Studio Knowledge Center for more information about licenses.

Search indices

In this area it exists search engine settings for all modules. Every module has a specific search index that stores all information that will be seachable. Rebuilding an index will create a lack of performance and should only be done by an administrator. To manually rebuild a search index, click on “Rebuild” next to current index. It will delete the old index totally and rebuild it from scratch. During the rebuild, information depending on the index will not be accessable. If you want the index to be accessable during rebuild, click "Update" next to a index. Old index will not be deleted.
