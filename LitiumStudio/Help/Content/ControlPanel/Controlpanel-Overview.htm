Control panel

The control panel contains all settings for all modules and the whole solution (Called system settings).

 

About this guide

 

See overview

This section describes how to reach the control panel.

 

See my settings

Information about how to configure Litium Studio for current user.

 

See system settings

This section describes briefly how to work with system settings such as search indicies, panels, or event log.

 

See Litium Studio Web Publishing

Settings specific for Web Publishing module, such as website texts, Google Analytics script, page types.

 

See Litium Studio E-Commerce

Settings specific for E-Commerce module, such as shopping cart settings, delivery providers, translations.

 

See Litium Studio Product Catalog

Contains all settings for Product Catalog. It exists settings for field templates, display templates, import articles.

 

See Litium Studio Media Archive

Settings specific for Media Archive, such as meta data or predefined versions.

 

See Litium Studio Relations

Contains all settings for Relations module. It exists settings for field templates or roles.

 

See Litium Studio Newsletter

Contains all settings for Newsletter module. It exists settings for templates or general settings.

 

Overview

The control panel is reached by clicking on Litium Studio start menu and select “Control Panel”.
