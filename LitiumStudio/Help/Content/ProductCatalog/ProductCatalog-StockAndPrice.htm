Warehouse

This area allows to you to create your warehouses that can be used when integrating to external stock balance systems.

Add warehouse
  1. Select Warehouse Node
  2. Click “Add”. You will be directed to the “Add warehouse” page.

Enter following information

  • Name - Name of the ware house
  • Description - Description of the warehouse
  • Is default website - Select this checkbox to make the warehouse as the default website
  • Address Line 1
  • Address Line 2
  • Zip
  • City
  • State
  • Country
  • Phone
  • Mobile
  • Fax
  • Email

 

Click “Save” to create new warehouse

Click “Cancel” to cancel creation of website

 

View warehouse details
  1. Select a warehouse from the list
  2. Click “View” and you will be directed to the view warehouse page

 

Click “Edit” to edit warehouse details.

Click “Back” to return.

 

Edit warehouse
  1. Select a warehouse from the list
  2. Click “Edit” and you will be directed to the “Edit warehouse” page
  3. Modify details.
  4. Click “Save” to save details

 

Delete warehouse
  1. Select the warehouse which you want to delete
  2. Click “Delete” and the selected warehouse/warehouses will be removed permanently

 

In order to import details

Import functionality allows you to import warehouse details as well as update its stock values. At present warehouse stock balances can only be updated using the warehouse xml file or through an integration.

 

  1. Click Import
  2. Click Browse and attach the relevant XML file
  3. Click “Import” Once successfully imported details (general details and stock balances) of the existing warehouses will be updated. If the warehouses referenced in the xml does not exist, new warehouses will be created with the specified products

Price lists

A price list consists of prices for a selection of articles, in a specific currency. The price list can target a specific segment of customers and can be valid during a specific time intervall.

 

When a price are displayed on the website, a calculation are made of every valid price list in current currency and the cheapest price are displayed.

Add price list

 

 

In order to add a price list.

  1. Click on the “Price list” (1) node found under “Stock and price” in the left menu.
  2. Select “New” found in the toolbar and select “Price list” in the drop menu.

Enter following details

  • Name - Name of the price list
  • Active - Select this checkbox to activate the price list Note: Only 'Active' price lists will be available for use in the application.
  • Currency - Currency used in the price list.
  • Valid interval - Valid time period of the price list. If you don't specify a valid interval, it will be considered valid for all dates
  • Valid for groups - User groups in Litium Studio Relation to which the price list is applicable to. If you don't specify a valid group, it will be considered valid to all users.
  • Valid for organizations - Organizations to which the price list is applicable for. If you don't specify a valid organization, it will be considered valid to all users.
  • Valid for product catalog - To which assortment this price should be applicable for. If you don't specify a valid product catalog, it will be considered valid for all product catalogs

 

Click “Save” to save the price list.

Click “Cancel” to abort changes made.

Click “Back” to return to the list of price lists.

Click “Edit”to edit the selected price list.

Edit price list

In order to edit price list details.

  1. Select “Price lists” (1) found in the left menu.
  2. Select a price list (2) to view and select “Edit” (b).

 

Click “Save” to save modifications done.

Click “Cancel” to abort changes made.

Delete price list

In order to delete a price list.

  1. Select “Price lists” (1) found in the left menu.
  2. Select a price list (2) to view and select “Delete” (c).
  3. Confirm that you actually want to remove the price list by clicking “Ok” to confirmation message that follows. This will remove the selected Price list permanently .
Price articles in price lists

In order to define price of articles in a price list.

  1. Select “Price lists” (1) found in the left menu.
  2. Select a price list (2) to view and select “Pricing” and click “Price articles” (e) given in drop down menu.

 

Enter following details

  • Price: Price.
  • Price includes VAT - If not checked, the total price will be “Price” x “VAT percentage”. If checked, the price will be the value that is entered and the price without VAT will be “Price” excluding “VAT percentage”.
  • VAT percentage - VAT percentage.

 

Add the articles that will have this price and VAT.

  1. Select all articles that should have this price, by clicking on the right arrow in the left article panel.

 

Click “Save” to save pricing information.

Click “Cancel” to abort changes made.

View priced articles

In order to define price of articles in a price list.

  1. Select “Price lists” (1) found in the left menu.
  2. Select a price list (2) to view and select “Pricing” and click “Priced articles” (d) given in drop down menu.
  3. Information about all priced articles will be displayed.

 

Click “Price articles” to add more articles to price list.

Click “Back” to return to the list of price list.

Export price list

A price list can be exported as XML. In order to do this:

  1. Select the required price list from the list
  2. Click Export (f).
  3. The XML-file can now be saved into your computer.
Import price lists

This feature allows you to import price lists from external sources. The file to import must meet the XML-format demands. See example below.

 

In order to import price lists.

  1. Click “Import” (g) found in the toolbar.
  2. In the next screen, select the XML-file that contains price list information. If a price lists exists with the same name as in the XML-file, it will be updated. If “Update articles” are selected, exactly the information that are provided in the XML-file will be shown in the price list. If the price list contains an article that are not provided in the XML-file, it will be removed.
