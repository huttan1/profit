Folders

Folders are very much similar to the hierarchical folder structures you will find in most applications. Folders allow you to create parent folders as well as sub folders and group your media accordingly. Significant feature of media archive folders is that you can set folder permissions for individual groups in order to provide the necessary folder security.

Create a folder

Follow these steps to create a new folder.

 

 

It exists two ways of creating a folder. One is to click on the “New” button in the toolbar, the new folder will be created under “Media Archive” node. The other way is to right click on the on the folder (1) will be parent to the new folder and click “new” (b).

 

Creation of a folder is done in the form of a wizard.

Create a folder - step 1

Enter following information

  • Name - Name of the folder

 

Click “To the next step” to go to Step 2

Create a folder - step 2

In this step you are required to set permission for the new folder. All groups created in relations will be given in the list box. Thus you can grant folder permission individually

Besökare (Visitors) and Redaktörer (Editors) are default user groups

In order to set permission
  • Select the required user group from the list
  • Specify if users in the selected group should be allowed to read and write by checking the Read, Write checkboxes. The permissions are used inside the administration as well as on the public website.

 

Click “Back to previous step” to return to step 1

Click “Save” to save folder details. Once saved the new folder will be added in folder structure.

Manage folders

 

 

 

  1. Add file(s) to selected folder.
  2. Create a new sub folder to selected folder.
  3. Edit selected folder.
  4. Delete selected folder and all files inside it.
  5. Move the selected folder upwards in the structure.
  6. Move the selected folder downwards in the structure.
  7. Move the selected folder to the top of current structure.
  8. Move the selected folder to the bottom of current structure.

 

It is also possible to drag and drop folders inside the structure.

 

Edit a folder
  1. Right click (1) on the folder, found in the folder structure.
  2. Select “Edit”(c) from the menu.

 

General tab

Enter following information

  • Name - Name of the folder
Permissions tab

In this step you are required to change permission for the folder.

 

Click “Save” to update the modifications.

Click “Cancel” to cancel the update.

 

Delete a folder
  1. Right click (1) on the folder, found in the folder structure.
  2. Select “Delete” (d) from the menu. The folder along with all files will be deleted.

Files and versions

General characteristic of media files is that they often have different versions and are subjected to constant upgrades. Different versions can be used for different purposes, such as one thumbnail version for the website, one version that are high definition, and one could be used for print.

 

 

Managing files

  1. View selected file
  2. Edit the selected file.
  3. Delete selected file(s).
  4. Cut the selected file
  5. Copy the selected file.
  6. Paste a file. Only visible if a file is cutted or copied.
Add file(s)

It exists two ways of adding new files. One way is to right click (1) on the folder and select “File”. The other way is to select a folder (1), click “New” in the toolbar and then select “File”.

 

Follow these steps to create a file-

Add file(s) - Step 1

Select the files that should be uploaded.

Select “Auto generate predefined versions” option to create predefined versions for the uploaded files. If selected, it will automatically generate versions using the predefined versions defined in Litium Studio Media Archive settings.

 

  • It is possible to upload multiple files.
  • Any file type can be uploaded at the same time.
  • The total size of selected files are limited. In the top of the page you will find information about what the current setting allows. That is configurated in web.config. Read more about it at Litium Studio Knowledge Center.

 

Click “To the next step” to proceed to Step 2 of the wizard. All selected files will be uploaded. This operation may take few minutes. Time taken for this operation will depend on the amount of files and their file sizes.

Click “Cancel” to abort changes made.

 

Add file(s) - Step 2

All files uploaded in the previous step will be displayed together with thumbnail view of the file, item name, file name and the file size.

Images will be displayed with a thumbnail picture. If is an ordinary file, file icon related to its extension will be displayed E.g. If it's a Microsoft Word document, a small word icon will be displayed. By default item name will be same as the file name. But you are allowed to modify the item name accordingly.

 

Click” To the next step” to proceed to Step 3.

Click “Back” to previous step to return to step

 

Add file(s) - Step 3

Step 3 of the wizard allows you to provide meta data for the attached media. Meta data is the data about data. In other words data that describes the specific media item. Once the metadata is added you can search your media files using these values.

In order to define metadata

Select a file from the list box

All metadata created from Media Archive setting will be displayed.

 

Use the text area given below Meta data fields to specify Metadata values.

Click to create new metadata. Two additional fields Name on new metadata and Value on new metadata will be added.

Name on new metadata refers to the name of the field and Value on New metadata refers to the value given for the particular metadata field. Specify values for these fields

If metadata has not been set up in Media Archive Settings you will be only be able to see a Create New metadata link. Click this link and define the values as given in 4. Click to remove the added Metadata field

 

Click “Back to previous step” to return to Step 2

Click “To the next step” to proceed to step 4 of the wizard

Add file(s) - Step 4

This step allows you to select the categories related to the uploaded files.

Select the relevant categories by checking the checkbox given next to the category name

 

Click “Back to previous step” to return to Step 3 of the wizard.

Click “Save” to save the folder.

This tab allows you to edit permissions set for the selected file. The selected file inherits permissions from its parent folder (permission set in Step 2 of Create New Folder Wizard. Refer Add New Media Archive Folder for more details)

Delete file(s)
  1. Select a folder (1) from the folder structure.
  2. Select a file to delete (2) and click “Delete“ (c). A confirmation box will appear, to verify that you want to delete this file.

Three options exists:

  • Check
    Check if the file are used by any other object in Litium Studio, such as a page or a product. All locations will be listed so it will be possible to replace the file with a new file.
  • Delete immediately
    The selected file will immediately be deleted from the category as well as from the folder.
  • Cancel
    Click to cancel the delete request

 

Before deleting media files it is always advisable to check if they are used in the application. If you delete a file that is currently used in the application that particular file will not be available for use. E.g. if used image in a web page is deleted from the media archive, the page will not show the image. A red cross will be shown instead. Once deleted the files/versions can't be restored later.

Managing versions

Create new version
  1. Select a folder from the folder structure.
  2. Select the file that you want to create a new version for and click “View”. Now select “Versions” from the toolbar and select “Create new version”

 

Creation of versions is done in the form of a wizard.

Create new version - Step 1

Enter following details:

  • Name - Name/version of the file to be uploaded.
  • How do you want to create the new version - It exists two options to create your new version:

    A resized version of the original file - The new version can be a resized version of your original image. This option are only available for images.

    Upload a new file - Allows you to upload a new file as version.

Click “To the next step” to proceed to step 2.

Click “Cancel” to return to the selected folder view.

Create new version - Step 2

Step 2 will be given depending on the option you selected for "How do you want to create the new version" in step 1.

  • “A resized version of the original file” - Specify the required height and width in the relevant text fields. These dimensions should be given in pixels. By default original height and width will be displayed. Specify if the aspects ratio of the file should be maintained by selecting the Keep aspects ratio of the file option. When this option is selected once you specify one dimension (height or width) other dimension will be automatically set considering the aspects ratio of the file
  • “Upload a new file” - Possibility to upload a new file as a version.

 

Click “Back to previous step” to return to step 1

Click” Save” to create the new version.

Create predefined versions
  1. Select a folder from the folder structure.
  2. Select the file that you want to create a new version for and click “View”. Now select “Versions” from the toolbar and select “Create predefined version”. All predefined versions will now be created for this file. The definitions for predefined versions can be found under Litium Studio Media Archive settings.

Click “Back to files” to return to the file list.

Remove resize versions
  1. Select a folder from the folder structure.
  2. Select the file that you want to create a new version for and click “View”. Now select “Versions” from the toolbar and select “Delete resize versions”. All versions will now deleted. If you want to delete a specific version, select a specific version when viewing the image. Then select “Delete version” found in the toolbar.

Click “Back to files” to return to the file list.
