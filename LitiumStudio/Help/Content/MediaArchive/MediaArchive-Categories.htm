Categories

As mentioned earlier Media Archive enables you to arrange your files in different folders and sub folders. Although your files are methodically placed in folders you might still have difficulty linking the related files. In such instances you could use categories. Categories allow you to associate files in several Litium Studio Media Archive folders.

Create category

Follow these steps to create an organization.

 

 

It exists two ways of creating a category. One is to click on the “New” button in the toolbar. The other one is to right click on the on “Category” (1) node and click “New”. (2)

Create a category - Step 1

Enter following details

  • Name - Name of the category

 

Click “Save” to save the category

Click “Cancel” to return to the category list

Manage files and categories

 

 

  1. View selected file
  2. Edit the selected file.
  3. Delete selected file(s).
  4. Cut the selected file
  5. Copy the selected file.
  6. Paste a file. Only visible if a file is cutted or copied.
  7. Delete selected file from current category. This will delete the selected items from the category but files in the folders will not be deleted.
Add a file to a category
  1. Select a folder from the tree and select a file from the file list.
  2. Click “Edit” (b) in the toolbar.
  3. Go to the “Categories” tab.
    All existing categories will be listed in the tab. Select the category from the list by clicking the checkbox given next to the category name.

 

Click “Save” to save changes made.

Click”Cancel” to abort changes made.

Edit a category
  1. Right click (1) on the category, found in the category tree.
  2. Select “Edit” from the menu.
General tab
  • Name - Name of the category.
Files tab
  • This tab lists all files in the category and also allows you to delete unwanted files from the category. In order to delete files from the category, select the checkboxes given next to the files that should be deleted and the click “Delete file from category” found in the toolbar. This will delete the selected items from the category but files in the folders will not be deleted.

 

Click “Save” to update the modifications. Deleted files will not be affected by this, they are deleted immediately.

Click “Cancel” to cancel the update.

Delete a category
  1. Right click (1) on the category, found in the category tree.
  2. Select “Delete” from the menu. The category will be deleted and the association with its files, but the files itselves will not be deleted.
Show category file
  1. Select a category (1) from the category tree.
  2. Select a file to view (2) and click “View“ (a).

 

Click “Save” to save changes made.

Click “Cancel” to abort changes.

Edit category files
  1. Select a category (1) from the category tree.
  2. Select a file to edit (2) and click “Edit“ (b).
General tab
  • Name - the name of the file.
  • New file - To replace the current file with a new file.
Metadata tab

Edit or add new metadata for current file.

Versions tab

Edit or add versions for the current file.

Categories tab

Define to what categories this file belongs.

Permissions tab

Set or edit group permissions for this file.

 

Click “Save” to save changes made.

Click “Cancel” to abort changes.

Delete category files
  1. Select a category (1) from the category tree.
  2. Select a file to delete (2) and click “Delete“ (c). A confirmation box will appear, to verify that you want to delete this file.

 

Three options exists:

  • Check
    Check if the file are used by any other object in Litium Studio, such as a page or a product. All locations will be listed so it will be possible to replace the file with a new file.
  • Delete immediately
    The selected file will immediately be deleted from the category as well as from the folder.
  • Cancel
    Click to cancel the delete request

 

Before deleting media files it is always advisable to check if they are used in the application. If you delete a file that is currently used in the application that particular file will not be available for use. E.g. if used image in a web page is deleted from the media archive, the page will not show the image. A red cross will be shown instead. Once deleted the files/versions can't be restored later.

Delete file from a category
  1. Select a category (1) from the category tree.
  2. Select a file to delete (2) and click “Delete file from category“ (g).

 

This option only deletes the selected file from the category. The original file in the folder will not be deleted.
