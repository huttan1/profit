Phone list

First step is to create your address list of phone numbers. When handling an site you may have to deal with large number of customers from different countries, industries, organization, etc. For your convenience it's better to create separate phone lists grouping customers who are most likely to get the same information from your organization. To make use of all customer information that are stored in Litium Studio Relations it is possible to connected an address list to several groups and organizations and the address list will always be up to date with address information.

Create phone list

Follow these steps to create an phone list.

 

 

 

  1. Select SMS node in the left tree and click on “Phone lists”.
  2. Click New and choose “Phone list”. You will be directed to “Create new recipient list” page. The creation of an phone list is done in a 3 step wizard.
Create phone list - Step1

At the first step you will have the possibility to to add phone numbers manually.

 

Enter following details.

  • Name: Name of the phone list.
  • Description: Brief description about the phone list and its contents.
  • Enter phone numbers:
  • Option 1: Enter numbers manually.
    Enter phone number of the recipients separated by a comma sign.
  • Option 2: Upload text file with phone numbers.
    Select a comma separated text file.
  • Click “To the next step” to proceed to the next step where it is possible to connect groups or organizations from Litium Studio Relations into your phone list.
  • Otherwise click “Save and close” to save phone list and exit the wizard.
Create phone list - Step 2

This step allows you to add phone numbers from persons connected to the selected groups to the address list. All groups created in Litium Studio Relations will be listed in the Available Group panel.

 

 

  1. In order to add a group to the phone list then select a group from the “Available Group” panel. Click on the right arrow to add a group.
  2. To exclude a group from the phone list, select the group from the “Synchronized groups” panel
  3. Click “Cancel” to abort all changes.
  4. Click “Save and Close” to save the phone list and exit the wizard.
  5. Click “Back to previous step” to return to Step 1.
  6. Click “To the next step” to proceed to Step 3 where you can add organizations.
Create phone list - Step 3

Step 3 enables you to add organizations to the phone list. All organizations created in relations will be given in the “Available Organizations” panel.

 

 

  1. In order to add an organization to the phone list, select an organization from the “Available Organization” panel by using the right arrow.
  2. To exclude an organization from the phone list, select the group from the “Synchronized organization” panel
  3. Click “Cancel” to abort all changes.
  4. Click “Save and Close” to save the phone list and exit the wizard.
  5. Click “Back to previous step” to return to Step 2.

 

Note: Once organizations are added, all persons in these organizations will be automatically be added to the email list.

Managing phone lists

 

 

View, edit and delete phone lists

  1. View selected phone list
  2. Edit the selected phone list.
  3. Delete phone list.
  4. Export all recipients from current phone list.
View phone list
  1. Select “Phone Lists” node (1) in the left tree. All lists currently available will be displayed.
  2. Select the list by selecting its row.
  3. Click “View” (a) and all recipients of the current address list will be displayed .
Edit general details
  1. Select “Phone Lists” node (1) in the left tree. All lists currently available will be displayed.
  2. Select the list by selecting its row.
  3. Click “Edit” (b) and the general information regarding that phone list will be displayed
  4. Click “Save” to save changes made or click “Cancel” to abort any changes made.
Edit receipt details
  1. Select “Phone Lists” node (1) in the left tree. All lists currently available will be displayed.
  2. Select the phone list (2) which you want to add new recipients.
  3. Click “View” (b) and all recipients will be displayed in a list.
  4. Select the person which you want to edit and click “Edit”. Click “Save” when finished with changes or “cancel” to abort changes made.
Add recipient to the phone list
  1. Select “Phone lists” node (1) in the left tree. All lists currently available will be displayed.
  2. Select the phone list (2) which you want to add new recipients.
  3. Click “View” (b) and all receipts assigned to current phone list will be displayed.
  4. Click “New recipient” in the toolbar.
  5. Enter phone number of the receipt.
  6. Click “Save” to save details or “Cancel” to abort changes made.
Delete receipt from phone list
  1. Select “Phone lists” node (1) in the left tree. All lists currently available will be displayed.
  2. Select the phone list (2) that you want to change.
  3. Click “View” (b) and all receipts in the selected list will be displayed.
  4. Select the recipient which you want to remove from the list.
  5. Click “Delete” to remove the selected recipient from the list. Or click “Delete All” to remove all the receipt from the list.
Import receipt to phone list
  1. Select “Phone lists” node (1) in the left tree. All lists currently available will be displayed.
  2. Select the phone list (2) that you want to import to.
  3. Click “View” (b) and then click “Import”.
  4. It exists two choices; either enter phone numbers manually or import using a comma separated file (csv)
  5. Click “Import” to start importing phone numbers or “Cancel” to abort changes made.
Manage synchronized groups and organizations
  1. Select “Phone lists” node (1) in the left tree. All lists currently available will be displayed.
  2. Select the phone list (2) that you want to change.
  3. Click “Synchronize” (d) and choose “Groups” or “Organizations” from the drop down menu.
  4. Groups and organizations that are already synchronized are shown in the right panel and groups and organizations that are possible to add are shown in the left panel. Use the right or left arrow to move items from one panel to another.
  5. Click” Save and Close” to save the modifications or “Cancel” to abort changes made.
Delete phone list
  1. Select “Phone lists” node (1) in the left tree. All lists currently available will be displayed.
  2. Select the phone list (2) that you want to change.
  3. Click “Delete”.
  4. Click “Ok” to the confirmation message that follows and the selected phone list will be deleted permanently

SMS

Create an SMS

Follow these steps to create a SMS

 

 

 

  1. Select “SMS and then select “All circulars” (1) in the left tree.
  2. Click “New” and choose “SMS” from the drop down menu. You wil nowl be directed to “Create new SMS” page. Creation of an SMS is done through a wizard.
Create an SMS - Step 1

In the first step, please add message of the SMS.

 

Click “To the next step” to proceed to Step 2 of the wizard.

Click “Save and Close” to save and exit the wizard. The circular will be saved as an uncompleted circular and will be moved to the Drafts folder.

Create an SMS - Step 2

In the second step is it possible to define information about who is sending this SMS and to who.

  • Sender Name *: Name specified as sender in Circular settings will be given here by default. You can change accordingly.
  • Recipient lists*: All available phone lists will be listed here. Select the list/lists which you want to send the phone.

 

Click” To the next step” to go to Step 3 of the wizard.

Click” Back to the previous step” to return to the previous step.

Click “Save and close” to save and exit the wizard. The circular will be saved as an uncompleted circular and will be moved to the Drafts folder.

Create an SMS - Step 3

Step 3 of the wizard provides a summary of the circular.

 

  • Preview circular: Enter an phone number and click “Send preview circular” to verify SMS information. A copy will immediately be sent to the provided phone number.
  • Click “Send” and the choose “Send immediately” to send the circular to the specified recipients instantly.
  • Click “Send” and then choose “Schedule sending” to schedule a time and date to send the mail.

 

Click ”Back to previous step” to return to Step 2.

Click “Save and close” to save and exit the wizard. The circular will be saved as an uncompleted circular and will be moved to the Drafts folder.

View and edit SMS

All incomplete SMS will be listed under the “Drafts” (b) node and completed emails will be listed under the “Sent” (c) node. Both emails that are sent and emails are incompleted will be listed under “All circulars” (a) node.

 

 

Drafts
View draft
  1. Select “Draft” (b) node. All uncompleted SMS will be listed in the right panel.
  2. Select the SMS which you want view.
  3. Click “View” (d) and details of the selected circular will be displayed to you
Edit draft
  1. Select “Draft” (b) node. All uncompleted SMS will be listed in the right panel.
  2. Select the SMS which you want view.
  3. Click “Edit” (e) and modify SMS details. Follow the same guide as when creating an SMS.
Delete draft
  1. Select “Draft”(b) node. All uncompleted SMS will be listed in the right panel.
  2. Select the SMS which you want delete.
  3. Click “Delete”(f).The selected SMS will be deleted permanently.
Copy draft
  1. Select “Draft”(b) node. All uncompleted SMS will be listed in the right panel.
  2. Select the SMS which you want copy.
  3. Click “copy circular” to create a copy of existing SMS
Sent

All SMS that are sent will be listed in this section.

View sent SMS
  1. Select “Sent” (c) node. All sent SMS will be listed in the right panel.
  2. Select the SMS which you want view.
  3. Click “View” (d) to see details about the selected circular
Copy sent SMS
  1. Select “Sent”(c) node. All sent SMS will be listed in the right panel.
  2. Select the SMS which you want copy.
  3. Click “copy circular” to create a copy of existing SMS. You will be taken to the “Copy existing circular” guide. The existing content of the selected circular will be copied to a new circular.

In this guide it is possible to specify how the existing recipients should be connected to the new circular. Available options are:

  • Refresh from the recipient list: This option will update the currently linked recipient list by including newly added recipients and remving deleted recipients from the list.
  • Only new recipients since original sending: This option will send the SMS to new recipients inserted to the list after the creation/sending of the original SMS.
  • Same recipient: This option will send the circular to the recipients included in the original SMS.
  • Select another recipient list: This option allows you to link the SMS to a new phone list. Once selected SMS list panel will be enabled. By default SMS lists linked to the original SMS will be selected. Select the SMS lists to which you want to send the copied SMS.

 

Click “Save and go to the circular list” to save the circular and return to the to the SMS list.
