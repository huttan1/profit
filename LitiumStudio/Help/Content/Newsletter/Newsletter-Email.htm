Address lists

First step is to create your address list of email addresses. When handling an site you may have to deal with large number of customers from different countries, industries, organization, etc. For your convenience it's better to create separate email lists grouping customers who are most likely to get the same information from your organization. To make use of all customer information that are stored in Litium Studio Relations it is possible to connected an address list to several groups and organizations and the address list will always be up to date with address information.

Create address list

Follow these steps to create an address list for emails.

 

 

  1. Select email node in the left tree and click on “Email lists”.
  2. Click New and choose Email list. You will be directed to Create New Recipient List page. The creation of an email list is done in a 3 step wizard.
Create address list - Step1

At the first step you will have the possibility to to add email addresses manually.

Enter following details.

  • Name: Name of the email list.
  • Description: Brief description about the email list /its contents.
  • Enter email address:
  • Option 1: Enter addresses manually.
    Enter email addresses of the recipients separated by a comma sign.
  • Option 2: Upload text file with email address.
    Select a comma separated text file.

 

The first step is now finished and it exists two possible ways.

  • Click “To the next step” to proceed to the next step where it is possible to connect groups or organizations from Litium Studio Relations into your address list.
  • Otherwise click “Save and close” to save email list and exit the wizard.
Create address list - Step 2

This step allows you to add emails from persons connected to the selected groups to the address list. All groups created in Litium Studio Relations will be listed in the Available Group panel.

 

 

  1. In order to add a group to the email list then select a group from the “Available Group” panel. Click on the right arrow to add a group.
  2. To exclude a group from the email list, select the group from the “Synchronized groups” panel
  3. Click “Cancel” to abort all changes.
  4. Click “Save and Close” to save the email list and exit the wizard.
  5. Click “Back to previous step” to return to Step 1.
  6. Click “To the next step” to proceed to Step 3 where you can add organizations.
Create address list - Step 3

Step 3 enables you to add organizations to the address list. All organizations created in relations will be given in the “Available Organizations” panel.

 

 

 

 

  1. In order to add an organization to the email list, select an organization from the “Available Organization” panel by using the right arrow.
  2. To exclude an organization from the email list, select the group from the “Synchronized organization” panel
  3. Click “Cancel” to abort all changes.
  4. Click “Save and Close” to save the email list and exit the wizard.
  5. Click “Back to previous step” to return to Step 2.

 

Note: Once organizations are added, all persons in these organizations will be automatically be added to the email list.

Managing email lists

 

View, edit and delete email lists

  1. View selected email list
  2. Edit the selected email list.
  3. Delete email list.
  4. Export all recipients from current email list.
View email list
  1. Select “Email Lists” node (1) in the left tree. All address lists currently available will be displayed.
  2. Select the list by selecting its row.
  3. Click “View” (a) and all recipients of the current address list will be displayed .
Edit general details
  1. Select “Email Lists” node (1) in the left tree. All address lists currently available will be displayed.
  2. Select the list by selecting its row.
  3. Click “Edit” (b) and the general information regarding that email list will be displayed Click “Save” to save changes made or click “Cancel” to abort any changes made.
Edit receipt details
  1. Select “Email Lists” node (1) in the left tree. All address lists currently available will be displayed.
  2. Select the email list (2) which you want to add new recipients.
  3. Click “View” (b) and all recipients will be displayed in a list.
  4. Select the person which you want to edit and click “Edit”. Click “Save” when finished with changes or “cancel” to abort changes made.
Add recipient to the email list
  1. Select “Email Lists” node (1) in the left tree. All address lists currently available will be displayed.
  2. Select the mail list (2) which you want to add new recipients.
  3. Click “View” (b) and all receipts assigned to current email list will be displayed.
  4. Click “New recipient” in the toolbar.
  5. Enter email address of the receipt.
  6. Click “Save” to save details or “Cancel” to abort changes made.
Delete receipt from email list
  1. Select “Email Lists” node (1) in the left tree. All address lists currently available will be displayed.
  2. Select the mail list (2) that you want to change.
  3. Click “View” (b) and all receipts in the selected list will be displayed.
  4. Select the recipient which you want to remove from the list.
  5. Click “Delete” to remove the selected recipient from the list. Or click “Delete All” to remove all the receipt from the list.
Import receipt to email list
  1. Select “Email Lists” node (1) in the left tree. All address lists currently available will be displayed.
  2. Select the mail list (2) that you want to import to.
  3. Click “View” (b) and then click “Import”.
  4. It exists two choices; either enter email addresses manually or import using a comma separated file (csv)
  5. Click “Import” to start importing email addresses or “Cancel” to abort changes made.
Manage synchronized groups and organizations
  1. Select “Email Lists” node (1) in the left tree. All address lists currently available will be displayed.
  2. Select the mail list (2) that you want to change.
  3. Click “Synchronize” (d) and choose “Groups” or “Organizations” from the drop down menu.
  4. Groups and organizations that are already synchronized are shown in the right panel and groups and organizations that are possible to add are shown in the left panel. Use the right or left arrow to move items from one panel to another.
  5. Click” Save and Close” to save the modifications or “Cancel” to abort changes made.
Delete e-mail list
  1. Select “Email Lists” node (1) in the left tree. All address lists currently available will be displayed.
  2. Select the mail list (2) that you want to change.
  3. Click “Delete”.
  4. Click “Ok” to the confirmation message that follows and the selected email list will be deleted permanently.

Email

Create a circular

Follow these steps to create a circular

 

 

  1. Select “All circulars” (1) in the left tree.
  2. Click “New” and choose “Email message” from the drop down menu. You wil nowl be directed to “Create new circular” page. Creation of a circular is done through a wizard.
Create a circular - Step 1

In the first step it will be possible to choose what template to use as email body. A template is the basic design and layout of a circular.

 

  • Select “Circular template” to use a specific template that are designed only for circulars. All templates that have been defined will be listed here. Define templates are done under settings for Litium Studio Circular.
  • Select “Page from Litium Studio” to use specific page as template. Choose page from a specific availiable.
  • In order to email a page from your website

 

Click “To the next step” to proceed to Step 2 of the wizard.

Click “Save and Close” to save and exit the wizard. The circular will be saved as an uncompleted circular and will be moved to the Drafts folder.

Create a circular - Step 2

In the second step it will be possible to work with mail content.

 

  • Subject: Enter the subject of the email here.
  • Content: If a page are selected you will only see a preview and are not able to change anything. If you did choose a “Circular template” at previous step, you will be able to work with the mail body.
  • Text Version : For email clients that does not accept HTML.

 

Click” To the next step” to proceed to Step 3 of the wizard.

Click “Back to the previous step” to return to step 1 of the wizard.

Click “Save and close” to save and exit the wizard. The circular will be saved as an uncompleted circular and will be moved to the Drafts folder.

Create a circular - Step 3

In the third step is it possible to define information about who is sending this email and to who

 

  • Sender Name *: Name specified as sender in Circular settings will be given here by default. You can change accordingly.
  • Sender email Address *: Sender email address specified in Circular settings will be given here by default. You can change accordingly.
  • Recipient lists*: All available email lists will be listed here. Select the list/lists which you want to send the mail.

 

Click” To the next step” to go to Step 4 of the wizard.

Click” Back to the previous step” to return to the previous step.

Click “Save and close” to save and exit the wizard. The circular will be saved as an uncompleted circular and will be moved to the Drafts folder.

Create a circular - Step 4

Step 4 of the wizard provides a summary of the circular.

 

  • Preview circular: Enter an email address and click “Send preview circular” to verify layout and information. A copy will immediately be sent to the provided email.
  • Click “Send” and the choose “Send immediately” to send the circular to the specified recipients instantly.
  • Click “Send” and then choose “Schedule sending” to schedule a time and date to send the mail.

 

Click ”Back to previous step” to return to Step 3.

Click “Save and close” to save and exit the wizard. The circular will be saved as an uncompleted circular and will be moved to the Drafts folder.

View and edit emails

All incomplete emails will be listed under the “Drafts” (b) node and completed emails will be listed under the “Sent” (c) node. Both emails that are sent and emails are incompleted will be listed under “All circulars” (a) node.

 

 

Drafts
View draft
  1. Select “Draft” (b) node. All uncompleted emails will be listed in the right panel.
  2. Select the email which you want view.
  3. Click “View” (d) and details of the selected circular will be displayed to you
Edit draft
  1. Select “Draft” (b) node. All uncompleted emails will be listed in the right panel.
  2. Select the email which you want view.
  3. Click “Edit” (e) and modify email details. Follow the same guide as when creating a circular.
Delete draft
  1. Select “Draft”(b) node. All uncompleted emails will be listed in the right panel.
  2. Select the email which you want delete.
  3. Click “Delete”(f).The selected email will be deleted permanently.
Copy draft
  1. Select “Draft”(b) node. All uncompleted emails will be listed in the right panel.
  2. Select the email which you want copy.
  3. Click “copy circular” to create a copy of existing email
Sent

All mails that are sent will be listed in this section.

View sent mail
  1. Select “Sent” (c) node. All sent emails will be listed in the right panel.
  2. Select the email which you want view.
  3. Click “View” (d) to see details about the selected circular

 

When viewing an email it is possible to view statistics about recipients. Click “Show recipients” in the toolbar.

 

Copy sent emails
  1. Select “Sent”(c) node. All sent emails will be listed in the right panel.
  2. Select the email which you want copy.
  3. Click “copy circular” to create a copy of existing email You will be taken to the “Copy existing circular” guide. The existing content of the selected circular will be copied to a new circular.

 

In this guide it is possible to specify how the existing recipients should be connected to the new circular. Available options are:

  • Refresh from the recipient list: This option will update the currently linked recipient list by including newly added recipients and remving deleted recipients from the list.
  • Only new recipients since original sending: This option will send the mail to new recipients inserted to the list after the creation/sending of the original mail.
  • Same recipient: This option will send the circular to the recipients included in the original mail.
  • Select another recipient list: This option allows you to link the mail to a new mail list. Once selected Email list panel will be enabled. By default email lists linked to the original mail will be selected. Select the email lists to which you want to send the copied mail.

 

Click “Save and go to the circular list” to save the circular and return to the to the email list.
