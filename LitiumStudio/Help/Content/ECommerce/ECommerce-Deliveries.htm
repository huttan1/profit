Deliveries

All deliveries are shown in the delivery list, found when selecting “Deliveries” in the left menu. It is also possible to select a range of deliveries by using a predefined filter, “Today”, “Yesterday, “This week”, “This month”, “This year” and “All.

Manage deliveries

 

  1. View the selected delivery.
  2. Delete selected delivery.
  3. Edit order status of selected order.
  4. View related order or payments associated with selected delivery.
  5. Send delivery information to distributor, if any are available.
  6. Filter delivery list based on a specified start date.
  7. Filter delivery list based on a specified end date.
  8. Filter delivery list based on a specified status.
  9. Filter delivery list based on a specified delivery method.
Add a delivery

It is possible to add a delivery to an existing order.

See Edit order. for more information

View a delivery
  1. Select a delivery row (1) from the list.
  2. Click “View” (a).

 

Click “Back” to return to the deliveries list.

Delete a delivery
  1. Select a delivery from the list (1).
  2. Click “Delete” (b). “Delete” might be disabled, depending on the delivery status settings.

Confirm that you actually want to delete the delivery from the list by clicking “Ok” to confirmation message that follows.

Edit delivery status

Edit status allows you to update the status of the delivery. Each of these statuses has a pre determined path defining the way they are used.

 

  1. Select delivery/deliveries (1) by ticking the checkbox given next to each row.
  2. Click “Edit status” (c). All available statuses for the selected delivery will be displayed in the drop down list. Delivery statues will be given depending on the predefined API rules.
View related payments and order
  1. Select the deliver (1) which you require to view related payments or order details.
  2. Click “View related” (d).
  3. Select order or payments from the drop down list. You will then be redirected to either order or deliveries associated with the delivery.
Send to distributor

Distribution of orders can be handled by delivery distributors or within your ERP. Send to distributor option available in Litium Studio facilitates communication between you and your delivery providers. Each delivery method can have a specific delivery provider.

 

The functionality of “Send to distributor” differs between delivery providers. Verify functionality with System Administrator.

 

In order to send notifications to distributors follow this guide:

  1. Select the delivery (1) you want to distribute.
  2. Click “Send to distributor” (e).
