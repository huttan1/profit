Order

An order is the foundation in Litium Studio E-Commerce and consists of different parts:

  • General information - The order date, currency for this order and order comments..
  • Order rows - One or many rows that consist of ordered products and their quanitity.
  • Customer information - Information about the customer.
  • Deliveries - Information about the deliveries, delivery method and fees.
  • Payment - Information about the payment, payment method and total price.

 

Add new order

New orders can be either be added by an order administrator inside Litium Studio E-Commerce or by a customer on the website.

 

 

Click “New” in toolbar and select “Order” (1).

General tab
  • Order date - This refers to the date which the order is placed. You are required to select the required date from the calender.
  • Currency - This is the currency in which the order will be placed. Select the required currency from the drop down. Additional currencies can be added in Litium Studio “Control panel”.
  • Comment - Add additional information related to the order.
Order rows tab

This tab allows you to add or edit order rows. An order row contain product information, quanitity and discounts connected to that order row.

 

 

Add order row

Follow this guide to add new items to current order.

 

  1. Click on “Add Row” (1) found in the toolbar.
  2. Enter following details:
    Product - Item to be added to the order row. Select the icon next to the textbox. All products of selected language that has a price will be shown.

    Quantity - Enter the quantity of the selected item.

    Discount - Discount given in the selected currency for the selected item.

    Comment - Additional information related to the order row.

 

Click “Add and close” to add the item to the order.

Click “Cancel” to bort changes made.

View order rows

In order to view details of an order row:

  1. Select the order row from the list.
  2. Click “View row” (b) found in the toolbar.

Details of the selected item will be displayed.

 

Click “Edit” to edit order row details.

Click “Back” to return to the order row list.

Edit a order row

In order to edit details of an order row:

  1. Select the order row from the list.
  2. Click “Edit row” (c) found in the toolbar.

 

  • Enter following details:
    Product - Item to be added to the order row. Select the icon next to the textbox. All products of selected language that has a price will be shown.

    Quantity - Enter the quantity of the selected item.

    Discount - Discount given in the selected currency for the selected item.

    Comment - Additional information related to the order row.

 

Click “Save” to save modifications.

Click “Cancel” to abort changes made.

Customer information tab

Customer information tab stores information about the customer who placed the order. You can either select an existing customer from Litium Studio Relations or enter it manually. Customer information will not be changed even if the information inside Litium Studio Relations is changed.

 

Following attributes are possible to enter:

  • First name
  • Last name
  • Customer number
  • Email
  • Mobile phone
  • Home phone
  • Fax
  • Address 1
  • Address 2
  • Zip
  • City
  • State
  • Country

 

Either enter values manually or add an existing person from Litium Studio Relations. Select a person by click on the icon next to “Person” text box and select a person. All text boxes will be populated with information from Litium Studio Relations. By selecting a person from Relations, it will be possible to track that a person is a e-commerce customer.

 

Once information are populated, click on “Deliveries” tab.

Deliveries tab

This tab enables you to specify delivery information of the order. An order can contain several

deliveries.

Add new delivery

In order to add a new delivery associated with an order:

  1. Click on “Add delivery” found in the toolbar.

 

 

  • Delivery address details
    Following attributes are possible to enter:

    First name
    Last name
    Customer number
    Email
    Mobile phone
    Home phone
    Fax
    Address 1
    Address 2
    Zip
    City
    State
    Country

    Either enter values manually, add the same address information as the one found under “Customer info” tab, or add an existing person from Litium Studio Relations. Select a person by click on the icon next to “Person” text box and select a person. All text boxes will be populated with information from Litium Studio Relations.

 

  • Delivery method - The method by which the product/products will be delivered to the customer. Select the preferred method from the drop down. Each method has a standard delivery cost that can be changed. Delivery method are defined in Litium Studio E-Commerce settings.
  • Delivery cost - Cost of the delivery.
  • Promised delivery date - The date on which the order are expected to be delivered to customer.
  • Delivery VAT % - VAT percentage changed for the delivery.
  • Delivery comments - Specify any comments related to the order delivery here.
  • Add order rows to current delivery - Select the check box, next to the order row that should be a part of this delivery.

 

Click “Add and close” to add the delivery row.

Click “Cancel” to abort changes made.

View delivery
  1. Under “Deliveries” tab, select the delivery which you want to view information about.
  2. Click “View delivery”.

 

Click “Back” to return to the delivery list

Delete delivery
  1. Under “Deliveries” tab, select the delivery which you want to delete.
  2. Click “Delete”. The selected delivery row will be deleted from the order. Depending on settings made for deliveries, it might be impossible to delete deliveries.

Order totals tab

Fields given in this tab will be automatically calculated by the application depending on the items ordered.

 

Available details

  • Order total - Sum of all order row.
  • Delivery cost -Sum of all delivery costs.
  • Fees - Fees applied on the whole order, for example payment fees.
  • Order discount - Discounts that are not associated with a campaign.
  • Campaign discount - Discounts that are a part of campaigns.
  • Grant total - Order total + fees + delivery cost - order discount - campaign discount + order VAT.
  • Includes VAT - The amount of VAT.

Managing orders

 

 

  1. View selected order.
  2. Edit selected order.
  3. Deleted seleted order.
  4. Copy the selected order.
  5. Edit order status of selected order.
  6. Export selected orders.
  7. View related deliveries or payments associated with selected order.
  8. Filter order list based on a specified start date.
  9. Filter order list based on a specified end date.
  10. Filter order list based on a specified status.
View order
  1. Select an order from the list (1).
  2. Click “View” (a).

 

In the following screen several actions are possible.

Click “Edit” to edit order details. “Edit” might be disabled, depending on the order status settings.

Click “Delete” to delete the selected order. “Delete” might be disabled, depending on the order status settings.

Click “Edit status” to edit the status of the order.

Click “Export” to export order information to XML format.

Click “Back” to return order list.

Click “Print” to print the order.

Edit order
  1. Select an order from the list (1).
  2. Click “Edit” (a). “Edit” might be disabled, depending on the order status settings.
Delete order
  1. Select an order from the list (1).
  2. Click “Delete” (c). “Delete” might be disabled, depending on the order status settings.

Confirm that you actually want to delete the order from the list by clicking Ok to confirmation message that follows.

Copy order

Copy order allows you to create a new order from an existing order.

  1. Select the order (1) to be copied from the list.
  2. Click Copy (d). The entire order will be copied to a new order. All data will be copied to the new order and the order number will be changed.
Edit order status

Edit status allows you to update the status of the order. Each of these statuses has a pre determined path defining the way they are used.

 

  1. Select order/orders (1) by ticking the checkbox given next to each order row.
  2. Click “Edit status” (e). All available statuses for the selected order will be displayed in the drop down list. Order statues will be given depending on the predefined API rules.
  3. Select status from the list.
Export orders
  1. Select a order (1) from the list.
  2. Click “Export” (f) and the order will be exported into XML. The format of XML-file can be changed.
View related payments and deliveries
  1. Select the order (1) which you require to view related payments or delivery details.
  2. Click “View related” (g).
  3. Select deliveries or payments from the drop down list. You will then be redirected to either payments or deliveries associated with the order.
Filter order list

To easier find related orders, it is possible to filter orders based on start date (h), end date (i) or order status (j).
