Overview

 

 

Toolbar

The toolbar area (A) contains all buttons and functions for Litium Studio E-Commerce. The content and which buttons that are enabled depends on the context.

Search area

The search area (B) will provide the administrator with a possibility to search for any searchable content in current module. Type any word in the search box and results will immediately be shown in the drop down list. The list will display one row per page that your search word are associated with. If “Enter” is hit on the keyboard, a specific search result page will be shown. Read more about search operations in the introduction chapter.

 

 

  1. The identification of the current item.
  2. A description of current item.
  3. “Show more” will display a specific search result page with all hits. Hit enter on the keyboard is the same as selecting this link.
Search result page

When selecting “Show more” or enter is hit, the search result page is shown. To the right of each page, it is possible to edit or view a page.

 

 

Left menu

The left menu (C) area consists of five parts:

  • Order management - Contains all orders in Litium Studio E-Commerce. Under this node it is possible to click on a date filter which will filter the order grid.
  • Deliveries - Contains all deliveries in Litium Studio E-Commerce. Under this node it is possible to click on a date filter which will filter the delivery grid.
  • Payments - Contains all payments in Litium Studio E-Commerce. Under this node it is possible to click on a date filter which will filter the payment grid.
  • Campaigns - Displays all campaigns in current solution.
  • Reports - Displays all reports created. In Litium Studio standard one report are included, however it is possible to include your own.

The width of the left menu can be changed by adjusting the splitter that separates the left menu from the content area.

Content area

The area (D) which displays all page specific information are called content area. The content inside this area will be based on what action that is taken in the left menu and toolbar.

The width of the content area can be changed by adjusting the splitter that separates the left menu from the content area.
