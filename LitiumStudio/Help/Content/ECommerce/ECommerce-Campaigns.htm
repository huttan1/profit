Campaigns

This section allows you to create online promotional offers. Litium Studio is distributed with

several predefined campaigns, but it is possible to define own campaigns with another functionality. Contact Litium or the reseller of Litium Studio.

 

Available campaign types are:

  • Buy X Pay Y: E.g. Buy 2 shirts and pay for 1.
  • X percent off: X % of the order total will be deducted when the buyer exceeds a 'Y' amount. E.g. 10 % of the order total will be deducted from orders exceeding SK10000.
  • Give Away: Once a product is purchased the customer will receive a free gift. E.g. Free toy given away with all baby products.
  • Free fees: Article/articles in this campaign will not be changed an Order Fee when the buyer exceeds an order total of 'X' amount.
  • Free Shipping: Article/articles in this campaign will be delivered FREE of charge when the buyer exceeds an order total of 'Y' amount.
  • X amount off: X amount will be reduced from total order price when the buyer exceeds an order total of Y amount. E.g. Campaign: SK 500 off .All purchases above Original SK 5000, will be given SK 500 reduction.
  • Voucher Single Use: A percentage or a defined amount X will be reduced when the buyer exceeds an order total of 'Y' amount. This campaign is done using vouchers and their voucher numbers are auto generated by Litium Studio. voucher codes generated for 'Voucher Single Use' campaign can only be used once .E.g. All purchases above SK 10000 will be given a 5% discount
  • Voucher Multi Use: A percentage or a defined amount will be reduced when the buyer exceeds an order total of 'Y' amount. This campaign is also done using vouchers. Unlike 'Voucher Single Use campaigns, Voucher Multi Use campaign vouchers need to be created manually. These vouchers can be used multiple times (number of times is defined in Step 2.E.g. All purchases above SK 10000 will be given a 5% discount
Create a campaign

Follow these steps to create a campaign.

 

  1. Select “All campaigns” found under “Campaigns” node from the left tree.
  2. Click “New” in the toolbar and select “Campaign” from the drop down menu. Creation of a campaign is done in the form of a 2 step wizard.
Create a campaign - Step 1

Enter following information:

  • Name - Name of the campaign
  • Description - Description of the campaign
  • Type - Select the campaign ty from the drop down. All campaign types supported by itium Studio will be given in the Type drop down.
  • Exclusive - There can be occasions where the same article is subjected to multiple campaigns (same article same valid period). In such instances exclusive campaigns will be given priority.
  • Active - Select this checkbox to activate the campaign. Once activated this particular campaign will be available in your ecommerce site. If left unselected it will be considered inactive and will not be shown in the ecommerce site.

 

Following fields will be displayed depending on the selected campaign type.

 

  • Valid for - Specify the time period (To date and from date) within which the campaign is applicable for.
  • Articles - Select articles applicable for the campaign.
  • Product groups - Select the product groups applicable for the campaign.
  • Product Catalogs - Select the product catalogs applicable for the campaign.
  • Groups in relations: Select the user groups which should benefit from this campaign. Note: All persons in the selected group/groups will benefit from the campaign.

 

Click “To the next step” to proceed to step 2.

Click “Cancel” to cancel the creation of the campaign and exit the wizard.

 

Create a campaign - Step 2

Step 2 of the wizard is dedicated to specifying campaign parameter values. Hence fields on this screen are dynamic and will depend on the selected campaign.

 

Click “Back to previous step” to return to Step 1.

Click “Save” to save campaign details.

View campaign details
  1. Select campaign to view.
  2. Click “View” in the toolbar.
Edit campaigns
  1. Select campaign to view.
  2. Click “Edit” in the toolbar.

 

Click “Save” to update modifications.

Click “Cancel” to abort changes made.

Copy campaign
  1. Select campaign to view.
  2. Click “Copy” in the toolbar.

 

You will be directed to the edit page of the selected campaign. If required change the details of the copied campaign. Copied campaigned will be added as Copy_Of_Campaing Name in the following manner.

 

Click “Save”.

Delete campaign
  1. Select campaign to delete.
  2. Click “Delete” in the toolbar.
  3. Click “Ok” to the confirmation message that follows. The selected campaign will be deleted permanently.
Activate/deactivate campaigns

Instead of deleting campaigns, they can be temporary deactivated during a period of time. This is also usefull if the administrator want to create campaigns before the activation date and kept in the pipe line to be activated in the future. All campaigns will only be active /valid till on or before the campaign end date and after that it will be deactivated automatically.

lthough a camping is activated it will only be applicable within the valid camping date range.

 

  1. Select the campaign which you want to activate.
  2. Click “Activate” or “Deactivated. The selected campaign will be activated or deactivated.
