Litium Studio Relations

User management of Litium Studio is handled by Litium Studio Relations. It handles relationship between persons, groups and organizations.

 

Note: You must have field templates for groups, organizations and persons before you can add a new item. (Field templates is created in “Control Panel” / “Relations” / “Field templates”.

Overview

 

 

Toolbar

The toolbar area (A) contains all buttons and functions for Litium Studio Relations. The content and which buttons that are enabled depends on the context.

Search area

The search area (B) will provide the administrator with a possibility to search for any searchable content in current module. Type any word in the search box and results will immediately be shown in the drop down list. The list will display one row per page that your search word are associated with. If “Enter” is hit on the keyboard, a specific search result page will be shown. Read more about search operations in the introduction chapter.

 

 

  1. The display name of current item. It can be the last name or group name.
  2. In this case it is the first name of current item.
  3. “Show more” will display a specific search result page with all hits. Hit enter on the keyboard is the same as selecting this link.
Search result page

When selecting “Show more” or enter is hit, the search result page is shown. To the right of each page, it is possible to edit or view a page.

Left menu

The left menu (C) area consists of three parts:

  • Persons - Contains all persons in current solution. Select “All person” to display a list.
  • Groups - Contains all groups in current solution. Select “All groups” to display a list.
  • Organizations - Under this node all organizations will be displayed. Next under “Organizations” the root organizations will be displayed along with its sub organizations.
  • Utilities - Contains common utilties for Litium Studio Relations. The node “Advanced search” will provide the user with possibility to do narrowed search.

 

The width of the left menu can be changed by adjusting the splitter that separates the left menu from the content area.

Content area

The area (D) which displays all page specific information are called content area. The content inside this area will be based on what action that is taken in the left menu and toolbar.

The width of the content area can be changed by adjusting the splitter that separates the left menu from the content area.
