Organizations

Create an organization

Follow these steps to create an organization.

 

 

Because an organization is structured hierarchical it needs to exists a parent before creating a sub organization.

 

It exists two ways of creating an organization. One is to click on the organization that will act as the parent organization (1) and click on the “New” button (2) in the toolbar. The other one is to right click on the organization that will act as parent organization for the new organization and click “new”. If it does not exists any organizations, click on the “New” button (2) in the toolbar and select “organization”.

Create an organization - Step 1

Enter following details

  • Template: Select the template which defines what information that can be used when creating this organization. Configuration of templates are found under settings for Litium Studio Relations.

 

Click “To the next step” to proceed to the next step

Click Cancel to exit the wizard

 

Create an organization - Step 2

In this step you will have the possiblity to enter general details about an organization. What kind of details that are visible, all depends on the configuration of current template.

  • All fields which are configured as “visible” and “required” will be mandatory.
  • All fields which are configured as “visible” will be displayed but will be considered as optional fields.
  • All field which are configured as “hidden” will not be displayed in the interface

 

Possible fields

  • Name - Name of the organization
  • Customer number - The customer number has to be unique.
  • Description - Description of the organization

 

Click “To the next step” to go to Step 3

Click “Back to previous step” to return to Step 1

Create an organization - Step 3

Enter information for custom fields as well as billing and delivery address.

  • All fields which are configured as “mandatory” must be give values.
  • All fields which are configured as “visible” and “uneditable” will be displayed but shown as read-only. Hence you will not be able to values for these fields
  • All field which you marked as “Hidden” will not be displayed in the interface
  • All fields which were marked as “Alternative” will be given in the form of a drop down

 

Click “To the next step” to proceed to Step 4.

Click “Save and finish” to save details and exit the wizard

Click “Back to previous step” to return to Step 2

Click “Cancel” to exit the wizard to Step 2

Create a organization - Step 4

Step 4 of the wizard allows you to assign persons to current organization

All currently existing persons will be listed in the “Available persons” panel

Assign persons to the organization
  • Select a person from the “Available persons” panel and click Add.
    The selected person will be added to the organization and will be listed in the “Members of the organization” panel.
Remove persons from the organization
  • Select a person from the “Members of the organization” panel and click “Remove”.
    The selected person will be removed from the organzation and will be moved to the “Available persons panel”

 

Click “Save and finish” to save the new organization

Click “Back” to previous step to return to step 3 of the wizard

Click “Cancel” to exit the wizard

 

Create a organization - Step 5

A person may perform single or even multiple roles for same/different organizations. Step 5 of the wizard is dedicated to mapping the relationship between associated organizations and roles. Same individual can hold different roles in the same organization

 

All availiable persons and roles are listed in two panels.

  • Select the person from “Availiable persons” panel which you want to assign a role to.
  • Select the role which the selected person are associated with.
  • Click “Add”. The newly created mapping will be displayed in “Member of roles” panel as shown below.

 

Click “Back to previous step” to return to Step 4 of the wizard

Click “Save and finish” to save the new Organization

Click “Cancel” to exit the wizard

Managing organizations

 

 

  1. Add persons to selected organization.
  2. Add a new organization as a sub organization.
  3. View selected organization.
  4. Edit selected organization.
  5. Delete selected organization.
  6. Cut the selected organization (To paste into another organization).
  7. Copy the selected organiation (To paste into another organization).
  8. Paste. (Will only be selected if you cutted or copied an organization).
  9. View selected persons
  10. Edit the selected persons.
  11. Delete selected persons(s).
  12. Manage person(s) in current organization.
  13. Import person(s) to current organization
  14. Export person(s) from current organization.
  15. Advanced search

 

To access the options of a-h, right click on the organization to work with. To access the rest, select the organization and the toolbar will be visible.

View organization
  1. Right click on the organization in the tree and select “View” (c).

Click “Edit” to edit the organization.

Click “Delete” to delete the organization.

Click “Back” to return back to the list of persons associated with the organization.

Edit organization
  1. Right click on the organization in the tree and select “Edit” (d).
General tab

This tab gives you general details of the organization; including all system and user defined fields.

Billing and shipping tab

Billing and shipping address details of the selected organization will be given in this tab.

Roles tab

All availiable persons and roles are listed in two panels.

  • Select the person from “Availiable persons” panel which you want to assign a role to.
  • Select the role which the person in selected organization are associated with.
  • Click “Add”. The newly created mapping will be displayed in “Member of roles” panel as shown below.
Persons tab

This tab allows you to modify persons attached to the selected organization.

 

In order to add new persons to the group

  • Select a person from the “Available persons” panel and click “Add”. The selected person will be added to the organization and will be listed in the “Members of the organization” panel

 

In order to remove existing persons from the group

  • Select a person from the “Members of the organization” panel and click “Remove”. The selected person will be removed from the organization.
Settings tab

Settings tab allows you to change the template used in creating the selected organization. All organization templates available in the system will be given in the template drop down. In order to change the template, select a suitable template from the drop down.

Note: Once the template is changed only the fields relevant to the new template will be displayed. Fields which is not available in the new template will be removed.

Click “Save” to save details

Click “Cancel” to abort changes made.

Delete organization
  1. Right click on the organization in the tree and select “Delete” (e).
Manage persons in the organization

Persons are generally assigned to an organization during its creation. But you can add/remove persons any time using the “Manage persons” option in the following manner.

  1. Right click on the organization in the tree and select “Manage person” (a).

You will be taken to the persons tab of the “Edit organization” page

 

Click “Save” to save modifications

Click “Cancel to cancel the modifications

Export persons

Export all persons to an CSV-file. All exported columns are separated by ; and each row represents by a person.

  1. Select the organization from which person are going to be exported from.
  2. Click “Export” (n) in toolbar.
Import persons
  1. Select the organization from which person are going to be imported to.
  2. Click “Import” (m) in toolbar.

Click Browse and attach the file to be imported. Follow the format of an exported file.

Specify following details

  • Groups: Select any of the following options
    Import without group reference
    If this option is selected imported persons will not be assigned to any group

    Import to group
    This option allows you to assign the imported persons to a particular group. All currently existing groups will be given in the Import to Group drop down. Select the group which you want to assign the imported persons to from here
  • Field Template
    Specify which Persons template should be used to import new persons (persons which do not currently exist in relations)

 

Existing person details (Based if the person ID exists) will be updated and new person details will be added as a new record

 

Click “Import and you will be redirected to a result page.

Click “Cancel” to abort changes made.

 

Advanced search

Advanced search allows you to create specific search queries.

  1. Select the organization to perform search from.
  2. Click “Advanced search” (o) in toolbar.

 

Specify one or more of the following options

  • Field Template
    Select the template which was used to create the searched person.
  • Group
    Select the group to which the person should belong to
  • Organization
    Select the organization to which the person below to
  • Role
    Select the tole which the selected person is holding
  • Fields (First Name, Middle Name, Last Name, etc)

 

Click “Search” to perform search and go to the search result page.

Click “Cancel” to abort changes made.

Search result

 

 

The search result will be presented as a list, the same layout that are used accross Litium Studio. The toolbar provides the same functionality as in every other part of Litium Studio Relations.

 

Every row provides two shortcuts to the right:

  1. Edit the information of current row.
  2. View information about current row
