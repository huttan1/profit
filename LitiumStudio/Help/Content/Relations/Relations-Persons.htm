Persons

A person contains personal information based on a dynamic field template so it is possible to store information in any format you like. A person holds login name, password and possibility for individual permissions.

Create a person

Follow these steps to create a person.

 

 

  1. Select “All person” found under “Persons” node from the left tree.
  2. Click “New” in the toolbar and select “Person” from the drop down menu. Creation of a person is done in the form of a 6 step wizard
Create a person - Step 1

Enter following details

  • Template: Select the template which defines what information that can be used when creating this person. Configuration of templates are found under settings for Litium Studio Relations.

 

Click “To the next step” to proceed to the next step

Click Cancel to exit the wizard

Create a person - Step 2

In this step you will have the possiblity to enter general details about a person. What kind of details that are visible, all depends on the configuration of current template.

  • All fields which are configured as “visible” and “required” will be mandatory.
  • All fields which are configured as “visible” will be displayed but will be considered as optional fields.
  • All field which are configured as “hidden” will not be displayed in the interface

 

Possible fields

  • First name
  • Middle name
  • Last name
  • Customer Number - Customer number given to the person. Has to be unique.
  • Title - Job ttile of the person
  • Gender - Female or male
  • Email
  • Company
  • Address 1
  • Address 2
  • Zip Code
  • City
  • Country
  • Image - Click “Browse” and select the picture of the person.
  • Comment - Additional comments about the person
  • Phone home
  • Phone work
  • Cell phone

 

Click “To the next step” to go to step 3

Click “Back to previous step” to return to step 1

Click “Cancel” to exit the wizard

Create a person - Step 3

Enter details about billing address, delivery address and additional fields.

  • All fields which are configured as “mandatory” must be give values.
  • All fields which are configured as “visible” and “uneditable” will be displayed but shown as read-only. Hence you will not be able to values for these fields
  • All field which you marked as “Hidden” will not be displayed in the interface
  • All fields which were marked as “Alternative” will be given in the form of a drop down

 

  • Billing address: Enter billing address here. If the person's billing address is same as personal address select Same as address checkbox given below the billing address panel to populate the address fields.
  • Delivery address: Enter delivery address here. If a person's delivery address is same as personal address select Same as address checkbox given below the Delivery address panel to populate the address fields.
  • Other Information: Custom fields added will be also displayed here

 

Click “To the next step” to proceed to Step 4.

Click “Save and finish” to save details and exit the wizard

Click “Back to previous step” to return to Step 2

Click “Cancel” to exit the wizard

Create a person - Step 4

Step 4 of the wizard allows you to assign the person to groups.

All existing groups will be listed in the “Available Group” panel

  • Select the group which you want connect and click “Add”. The added group will be moved to the “Member of groups” panel.
  • To remove a group, select a group from the “Member of groups” panel and click “Remove”

 

Click “To the next step” to proceed to step 5.

Click “Save and finish” to save details

Click “Back to previous step” to return tosStep 3

Click “Cancel” to exit the wizard

Create a person - Step 5

Step 5 of the wizard allows you to assign the person to related organization/organizations.

 

All existing organizations will be listed in the “Available organizations” panel

  • Select the organization which you want assign and click “Add”. The added organization will be moved to the “Member of organizations” panel.
  • To remove an organization, select an organization from the “Member of organizations” panel and click ”Remove”.

 

Click “To the next step” to proceed to Step 6.

Click “Save” and finish to save details

Click “Back to previous step” to return to Step 4

Click “Cancel” to exit the wizard

Create a person - Step 6

A person may perform single or even multiple roles for same/different organizations. Step 6 of the wizard is dedicated to mapping the relationship between associated organizations and roles. Same individual can hold different roles in the same organization

 

All availiable organizations and roles are listed in two panels.

  • Select the organization from “Availiable Organizations” panel which you want to assign a role to.
  • Select the role which the person in selected organization are associated with.
  • Click “Add”. The newly created mapping will be displayed in “Member of roles” panel as shown below.

 

Click “Save and finish” to save details.

Click “Back to previous step” to return to Step 5

Click “Cancel” to exit the wizard

Managing persons

View, edit and delete persons

 

 

  1. View selected person
  2. Edit the selected person.
  3. Delete selected person(s).
  4. Assign selected person(s) to groups and organizations.
  5. Export selected person(s).
View person
  1. Select “All person” (1) found under “Persons” node from the left tree.
  2. Select person to view (2).
  3. Click “View” (a) in toolbar and all details regarding current person will be displayed.

 

Click “Edit” to edit the person.

Click “Delete” to delete the person.

Click “Back” to return back to the person list.

Edit person
  1. Select “All person” (1) found under “Persons” node from the left tree. It is also possible to select a person when viewing a group or an organization.
  2. Select person to edit (2). Click “Edit” (b) in toolbar.
General tab

All details (except billing and shipping details) given during creation of the person will be shown in this tab.

Billing and shipping tab

Billing and shipping address details of the selected person will be given in this tab.

Groups tab

This tab displays the groups which the selected person is assigned to.

All existing organizations will be listed in the “Available organizations” panel

  • Select the organization which you want assign and click “Add”. The added organization will be moved to the “Member of organizations” panel.
  • To remove an organization, select an organization from the “Member of organizations” panel and click ”Remove”.
Organizations tab

This tab displays the organizations which the selected person is assigned to.

All existing organizations will be listed in the “Available organizations” panel

  • Select the organization which you want assign and click “Add”. The added organization will be moved to the “Member of organizations” panel.
  • To remove an organization, select an organization from the “Member of organizations” panel and click ”Remove”.
Roles tab

All availiable organizations and roles are listed in two panels.

  • Select the organization from “Availiable Organizations” panel which you want to assign a role to.
  • Select the role which the person in selected organization are associated with.
  • Click “Add”. The newly created mapping will be displayed in “Member of roles” panel as shown below.
Litium Studio login tab

Litium Studio Tab allows you to manage username, password and permissions.

 

User credentials

  • User name
    The user name that current person will use when signing in.
  • Password
    The password to use. It might exist a password policy that are restricting password length or password format. Contact system administrator for more information.
  • Account locked
    If an account is locked (I.e. because of too many login attempts), it is possible to unlock the account. For instance, it is possible to lock an account manually.
  • User has to change password next time he or she logs in
    Use this function to require that the person has to change password at the initial login (once the username and password has been created/ modified).

 

Permissions

In order to set overall permission select the following options accordingly

  • All
    Select this option to give rights to the entire application. This type of user will have full access to all Litium Studio modules and their functionality.
    To insure security of the application it is advisable that you only give full module access to your system administrator
  • User handling
    Select this option to give access to users and groups settings found under the control panel.
  • Language handing
    Select this option to give access rights to “Language settings”.

 

Module permission

Select the module which you require to set permission for

Set permission according to the user role:

  • All
    Select this option to give full rights to the selected module. A person with full access will be able to perform all functionality related to the module. To insure security of the module it is advisable that you only give full module access to your system administrator or module administrator
  • Access to Module
    This option will give the person access to the module but needs specific permissions to access something else. I.e. a person that should only work with the english website, needs this permission along with web site permissions.
  • Handle all content
    This option will allow the person to manage all content in the selected module
Settings tab

Settings tab allows you to change the Persons' template used in creating the selected user. All persons templates available in the system will be given in the template drop down. In order to change the template, select a suitable template from the drop down.

Note: Once the template is changed only the fields relevant to the new template will be displayed. Fields which is not available in the new template will be removed.

 

Click “Save” to save details

Click “Cancel” to abort changes made.

Delete person
  1. Select “All person” (1) found under “Persons” node from the left tree.
  2. Select person to delete (2).
  3. Click “Delete” (c) in toolbar. You will receive an alert where you confirm that you want to delete current person(s).
Assign groups/organizations

A quick way to assign multiple persons to several groups or organizations.

 

  1. Select “All person” (1) found under “Persons” node from the left tree.
  2. Select person(s) to assign to groups/organizations (2).
  3. Click “Assign groups/organizations” (d) in toolbar.

 

You will be directed to assign persons to groups/organizations page

All available groups and organizations will be listed in the “Available group” and “Available organization” panels respectively.

  • In order to assign users to groups/organizations
    Select a group/organization from the relevant panel and click “Add”
  • If you need to remove any group/organization
    Select group/organization to be removed and click remove

 

Click ”Save” to save modifications

Click “Cancel” to abort the modifications

Export persons

Export all persons to an CSV-file. All exported columns are separated by ; and each row represents by a person.

  1. Select “All person” (1) found under “Persons” node from the left tree.
  2. Select person(s) to export (2).
  3. Click “Export” (e) in toolbar.
Import persons
  1. Select “All person” (1) found under “Persons” node from the left tree.
  2. Click “Import” (e) in toolbar.

 

Click Browse and attach the file to be imported. Follow the format of an exported file.

Specify following details

  • Groups: Select any of the following options
    Import without group reference
    If this option is selected imported persons will not be assigned to any group

    Import to group
    This option allows you to assign the imported persons to a particular group. All currently existing groups will be given in the Import to Group drop down. Select the group which you want to assign the imported persons to from here
  • Field Template
    Specify which Persons template should be used to import new persons (persons which do not currently exist in relations)

 

Existing person details (Based if the person ID exists) will be updated and new person details will be added as a new record

 

Click “Import and you will be redirected to a result page.

Click “Cancel” to abort changes made.

Advanced search

Advanced search allows you to create specific search queries.

  1. Select “All person” (1) found under “Persons” node from the left tree.
  2. Click “Advanced search” (e) in toolbar.

 

Specify one or more of the following options

  • Field Template
    Select the template which was used to create the searched person.
  • Group
    Select the group to which the person should belong to
  • Organization
    Select the organization to which the person below to
  • Role
    Select the tole which the selected person is holding
  • Fields (First Name, Middle Name, Last Name, etc)

 

Click “Search” to perform search and go to the search result page.

Click “Cancel” to abort changes made.

Search result

 

 

The search result will be presented as a list, the same layout that are used accross Litium Studio. The toolbar provides the same functionality as in every other part of Litium Studio Relations.

 

Every row provides two shortcuts to the right:

  1. Edit the information of current row.
  2. View information about current row
