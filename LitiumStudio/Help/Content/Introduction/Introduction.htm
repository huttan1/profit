Introduction

What is Litium Studio

Litium Studio is one of the Nordic area's premier CMS suppliers and market leader with combining CMS and ecommerce. With Litium Studio you manage all aspects of you web site - from web publishing to newsletters, product publishing and ecommerce.

Litium Studio comes with six modules for different purposes. Which modules that you have access to depends on the license.

 

 

Litium Studio Web Publishing

Adaptable and effective information management, visitor statistics, multi-language support, smart template files, search engine adjustment and lots of accessories and modules.

 

Litium Studio Product Catalog

Easily handles thousands of products for automatic web publishing, e-commerce and digital product catalogs. Complete language support and easy integration with ERP systems.

 

Litium Studio Ecommerce

Complete and customizable ecommerce for both B2C and B2B, shopping cart, customer profile, payment solutions and check-out flow.

 

Litium Studio Relations

Manage and develop your audiences on the Web; logins, organizations, roles, membership history, customer history, driven publishing and targeted mailings.

 

Litium Studio Media Archive

Professional management of the organization's all pictures and documents. Scaling, search, cataloging, and access control for web, presentations, product catalog and press.

 

Litium Studio Newsletter

Customizable e-mail to thousands of recipients with follow-up, layout templates and the ability to send web pages from the site.

Getting started

Once the installation is complete you need to sign in into the application using the system administrator username and password established during installation.

In order to use the system the system administrator has to sign in into the system and set up other user accounts.

To sign in
  1. Type '/LitiumStudio' after the domain name. You will be taken to the Litium Studio login page. E.g. http://www.litiumstudio.se/LitiumStudio
  2. Enter the username and the password of current user
  3. Click “Log in”.

 

 

 

You will now be redirected into Litium Studio E-Commerce or Litium Studio Web Publishing depending on solution settings.

Common functionality
Filter area

In several places in Litium Studio it exists a filter area with contains functionality to work with the content of current view. For example inside the product catalog it exists a filter area to narrow down the article lists. Immediately when selecting a filter option, the list will be reloaded according to the filter options.

 

 

 

 

 

List view

In places with large amount of data it exists a list view, which will provide the user with possiblity to sort and navigate from page to page.

 

Columns in a list view

To sort a column in a list view, click on the column header to with you want to sort. The column which are sorted are blue and has a arrow icon next to it. If the arrow is pointing upwards, it is sorted ascending. In this case the “Article number” are sorted ascending.

 

 

Navigation bar

When a large amount of data is presented, a navigation bar will be shown at the bottom of this list view.

 

 

  1. Click on this button to go to the previous page.
  2. Select the slider and drag it to prefered page.
  3. Click on this button to go to the next page.
  4. Displays the total amount of pages and on the page number of current page.
  5. Displays the total amount of records and the current amount of record on this page.

 

Litium Studio search

Litium Studio search engine is a free text engine which acts similar to other search engines, such as Google. Search is easy, just type the words that you want to search for and click "Search". Examples are denoted using [] around the search terms, for example [Apple iPhone].

 

General information

  • Search is always case insensitive.
  • Every word in a search query matters.
  • Litium Studio has a built in logic to find matches to a search term, even though it is misspelled.
  • File name and file content are searchable if search components are installed. Please visit Litium Studio Knowledge Center for more information (kc.litiumstudio.se).

 

Advanced search queries

  • Phrase search ("")
    Using double quotes around words will match search results which contains those words in exactly the same order.
    Example: ["Apple iPhone"] will match content exactly in that order.
  • Exclude word (-)
    A minus sign before a word will exclude that word from the search query.
    Example: [Apple -iphone] will match content containing Apple but not iPhone.
  • Wild cards (*)
    A wild card will fill in the blanks in a search query. It can be used inside or after a word.
    Example: [App*] will match Apple or AppStore.
