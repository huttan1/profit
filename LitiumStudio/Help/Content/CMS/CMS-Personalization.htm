Personalization

My tasks

My task lists the task pending /related to the signed in user. In order to access this section Select “My tasks”. An overview will be shown with the five latests items.

 

In the page list, the page name and its corresponding website will be displayed. Next to each page, two icons are shown. One to edit and one to view a page.

 

 

My working copies

All pages which have been modified, by the signed in user, but not published will be shown here.

 

Pages to approve and publish

Pages that are a part of approval work flow, for the signed in user, will be shown here. A page are a part of a apporval work flow if it is published as “Ready for publishing”.

Any authorized user will see these pages and have the possiblity to approve them.

 

My page responsibility

All pages which were created selecting the signed in user as the responsible person will be listed here.

 

My translations

All translated pages which were created selecting the signed in user as the responsible person will be listed here.

 

My note

Here you can post notes related to things you should remember.

 

Add note

In order to add new notes

  1. Select “My note” node from the tree.
  2. Click “New note”.
  3. Enter brief description of the note.

 

Click “Save”.

 

Edit note

In order to edit a note.

  1. Select “My note” node from the tree.
  2. Click on the edit icon the right of selected row.
  3. Enter brief description of the note.

 

Click “Save”.

 

Delete note

In order to delete a note.

  1. Select “My note” node from the tree.
  2. Click on the delete icon the right of selected row.
  3. Click ok to the confirmation that follows
